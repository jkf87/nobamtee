# concrete-essay-ko PROVENANCE

- 유형: 디폴트 프리셋 (저장소 동봉, 배포 가능)
- register: 칼럼·에세이·문학 단편. 짧은 감각적 단문 위주의 한국어 산문.
- `fingerprint.json`: 본 register 의 목표 지표값. 손으로 추정한 합의값.
- `descriptor.md`: 본 저장소 직접 작성, 사람 큐레이션.
- `exemplars.md`: 본 저장소 직접 작성. 일반 주제 4편.
- 라이선스:
  - `fingerprint.json`: 사실 데이터, CC0.
  - `descriptor.md`·`exemplars.md`: 본 저장소 원저작물, CC0.

## 프리셋 선택 가이드

칼럼·에세이·문학 단편처럼 짧고 감각적인 산문 register 에 사용한다. 기술 글에는 `plain-tech-ko` 를 권장.
